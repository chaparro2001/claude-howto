# Generador de Documentacion API

Generar documentacion API mediante:

1. Escanear todos los archivos en `/src/api/`
2. Extraer firmas de funciones y comentarios JSDoc
3. Organizar por endpoint/modulo
4. Crear markdown con ejemplos
5. Incluir schemas de request/response
6. Agregar documentacion de errores

Formato de salida:

- Archivo Markdown en `/docs/api.md`
- Incluir ejemplos con curl para todos los endpoints
- Agregar tipos TypeScript
